Introduction to JavaScript

JavaScript ― это интерпретируемый язык программирования общего назначения с динамической типизацией данных.

Интерпретируемый значит что программа выполняется в исходном виде, т.е., исходный код не преобразуется в машинный или промежуточный код. В связи с этим ошибок на этапе компиляции и компоновки отсутствут, есть только ошибки на этапе выполнения. FrontEnd

Диамическая типипзация говорит о том, что тип переменных и констант определяется во время присваивания, по типу значения справа (r-value). l-value = r-value;

Переменные и типы данных

Переменные и константы

var
объявляет глобальную переменную, несмотря на область видимости, в которой она объявлена. Если записать значение в несуществующую переменную, то это не вызовет ошибку, это будет аналогично ключевому слову 'var';
let
объявляет локальную переменную, только для той области видимости, в которой она используется;
const
объявляет константу. Константы могут быть только локальными;
